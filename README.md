# WebAPIwithASP.NET
## Web API with ASP.NET I did for my school course

You will need:
```
Postman
```


Postman:
```
Post: 	
http://localhost:5000/api/todo/

{
  "name":"Enter name here",
  "isComplete":true
} 

Put:    
http://localhost:5000/api/todo/{id}

{
  "id":2,
  "name":"hehe",
  "isComplete":true
}

Delete:
http://localhost:5000/api/todo/{id}

```
